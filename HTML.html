lesson 1: basic html structure

every single html file you create requires a fundamental layout skeleton so the browser knows how to parse and render the text correctly. here is what a clean template looks like:

<!DOCTYPE html> <html> <head> <title>page title</title> </head> <body> <h1>hello world</h1> </body> </html>
