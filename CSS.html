lesson 1: css selectors and colors

css is what brings your structural html layers to life using layouts, margins, and presentation properties. to change how an element looks, you target it using a class selector (written with a dot prefix):

/* targeting an html element by its class name */ .login-btn { background-color: #38bdf8; color: #0f172a; border-radius: 8px; }
